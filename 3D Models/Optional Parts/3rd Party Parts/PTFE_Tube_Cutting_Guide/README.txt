# Summary

Cut your PTFE tube nice and square using a utility knife or razor blade. Gives a much better cut than using flush cutters.

The opening for the blade is 1mm wide.

Update: you can now open this in Thingiverse Customizer if you want to change the size of the PTFE tube or blade opening.
